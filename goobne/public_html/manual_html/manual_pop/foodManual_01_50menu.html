  • 굽네 페퍼로니 찹찹피자 조리 메뉴얼(영상)

1. 주재료

1) 페퍼로니 찹찹 피자 냉동피자 (냉동유통)
포장단위 : 610g/ea
청구단위 : 10ea/1box
유통기한 : 제조일로부터 냉동 6개월
- 냉장 해동 : 냉장 1장 3시간/ 3장 6시간/ 5장 8시간
- 전자렌지 해동 : 6분40초 (기존동일)
- 단체주문 해동 : 12분 (메뉴얼 기존 동일)

2) 버터갈릭 소스 (냉장유통)
포장단위 : : 500g pack
청구단위 : 1pack
유통기한 : 제조일로부터 냉장 3개월
- 소분 사용시 소스통에 담긴 제품 7일 이내 사용
- 소스통에 소분하고 팩에 남겨진 소스는 30일 이내 사용

3) 굽네 눈꽃치즈 (냉동유통)
포장단위 : : 100g/ea
청구단위 : : 1pack(4ea)
유통기한 : 제조일로부터 냉동 12개월
- 개봉 전 : 냉동 보관 / 개봉 후 : 냉장 보관 7일 이내 사용
- 냉장 해동 : 냉장 1시간 이상

2. 기구 및 장비

1) 위생장갑 2) 가위 3) 그리드 4) 오븐장갑 5) 피자 커팅보드 6) 반달칼 7) 피자 삽
8) 전자레인지

3. 제조 (굽기)

1)유통기한 / 선입선출/ 제품에 이상 유무 확인 한 후 제품 조리 준비한다. 피자를 그리드/타공팬 올린 후 비닐을 제거 한다.
<전자레인지 해동 시 분무기 이용 물 도포를 하지 않는다.>

2)가위를 이용 하여 피자 테두리의 탄 부분을 정리한다.

3)엣지 부분에 있는 페퍼로니와 모짜렐라 치즈가 있을 경우 피자 안쪽으로 토핑 정리를 해준다.

4) 소스통을 이용하여 엣지 부위에 버터갈릭 소스 20g을 동그랗게 도포한다.
<엣지의 딱딱해지는 식감을 개선 하며, 부드럽고 쫄깃한 식감, 엣지 끝까지 맛있게 먹을 수 있는 목적>
<분무기 이용 물 도포 하지 않는다>

4. 제조 (구운 후 토핑 및 포장)

5) 오븐 온도가 195℃ 되었는지 확인 후 오븐 조리
- Combi Master Plus : 7~9분 조리
- Combi Master 구형오븐 : 7분 조리 후 그리드를 반대로 돌려 2~3분 더 조리 (돌리지 않으면 한쪽만 진하게 조리됨)
- Self Cooking Center : Combi Master Plus 보다 1분정도 더 조리. 8~10분
- <오븐 상단에서 조리하여 2차 오염 방지>
 

6) 오븐 조리된 제품을 컷팅 보드에 옮긴 후 컷팅 휠,반달칼을 이용하여 8조각 되게 컷팅 한다.
<컷팅 한 사이로 기름이 흘러 내릴 수 있음. 컷팅 보드는 수시로 닦아 사용>

7) 피자박스에 유산지를 깔고 담은 후 굽네눈꽃치즈 10g을 골고루 뿌려준 후 삼발이로 고정한다.

8) 박스 포장 후 피자 봉투에 오이피클 갈릭디핑소스, 핫소스 각 1개씩 담아 제공한다.
<페퍼로니 찹찹 피자 홀매장 전용 완제품>

*주의사항

< 참조 >
  • 1. 오븐 조리 시 페퍼로니 피자에서 나오는 기름 발생으로 인하여
    • 버터갈릭 엣지 쪽으로 흘러 형태가 흐트러 질수 있으나 문제 없음.
  • 2. 피자에 치킨 향이 밸 수 있으므로 기름받이 물을 자주 갈아줄 것
  • 3. 종이 호일을 깔고 조리 시 피자 도우의 조리가 덜 될 수 있고 밑면이 눅눅해질 수 있음
  • 4. 피자를 상단에 조리하여 바람에 치즈가 날려 치킨 위에 떨어질 수 있으므로 주의 필요 (아주 소량이고 미미하지만 한 번 더 확인이 필요)
  • 5. 치즈가 굳을 시 재조리 하여도 녹지 않고 도우가 딱딱해질 수 있으니 재조리 금지
  • 6. 유산지 위에서 피자를 커팅 할 경우 종이가 잘려 피자에 혼입 우려 있으므로 금지
  • 7. 지형이 험한 곳으로 배달 시 피자가 쏠릴 수 있으므로 삼발이 세 개 정도로 고정 추천
  • 8. 피자 해동 시 해동 모드가 아닌 조리 모드 사용 시 과 조리되어 품질이 저하될 수 있으므로 해동모드 매뉴얼 준수하여 해동할 것
