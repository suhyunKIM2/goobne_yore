1. 주재료

1) 바비큐 시카고 딥디쉬 피자 (냉동유통)
포장단위 : 700g/ea
청구단위 : 8ea/1box(500*500*100)
유통기한 : 제조일로부터 냉동 6개월
- 냉장 해동 : 냉장 10시간 이상
- 전자렌지 해동 : 6분40초 (기존동일)
- 단체주문 해동 : 12분 (매뉴얼 기존 동일)

2) 그레이티드 파마산 치즈가루(냉장유통)
포장단위 : 227g/ea / 청구단위 : 1ea
청구단위 : 1pack
유통기한 : 별도 표시일 까지
- 매장 입고 시 냉장 보관/ 개봉 후 냉장 보관

2. 기구 및 장비

1) 위생장갑 2) 가위 3) 그리드 4) 오븐장갑 5) 피자 커팅보드 6) 반달칼 7) 피자 삽
8) 전자레인지

3. 제조 (굽기)

1)유통기한 / 선입선출/ 제품에 이상 유무 확인 한 후 제품 조리 준비한다.

2)피자를 그리드 or 사각토스트 팬에 올린 후, 가위를 이용하여, 비닐을 제거한다.

3)오븐 온도가 195도 되었는지 확인 후, 오븐 상단에서 9~11분 구워 준다.
<치킨과 함께 조리 시 제품에 기름이 떨어 지지 않도록 최상단 에서 구워준다.>

4)컷팅 보드에 구워진 피자를 올리고 반드시 반달칼을 이용하여 8조각 되게 컷팅 한다.
<컷팅 휠 사용 불가 (사용시 제품손상 우려)>

주의 사항 ① 구워진 피자는 8조각으로 컷팅 주의 사항 ② 반드시 컷팅 후 , 파마산 치즈를 올려 주세요. 주의 사항 ③ 조리 20분 이후 치즈는 조금씩 굳어갑니다.

4. 제조 (구운 후 토핑 및 포장)

5) 시카고 전용 박스에 엠보싱지를 깔고, 컷팅 한 피자를 담는다.
( 컷팅 시 엣지가 부서진 경우 잘 모아 동그란 형태가 되도록 담는다.)

6) 파마산 치즈 4g를 전체적으로 골고루 뿌려준 후, 삼발이로 고정한다.

7) 박스 포장 후 피자 봉투에 오이피클, 갈릭디핑소스 , 핫소스 각 1개씩 담아 함께 제공 한다.

*) 홀 매장 전용 제품 완제품

* Combi Master 구형 오븐 사용 주의사항

1) Combi master 구형 오븐 사용 시 5분 조리 후 그리드를 반대로 돌려 4~6분을 더 조리 한다.

( 피자조리 시 신형 Combi master Plus 오븐 사용 권장)

* 피자 컷팅 TIP

1) 오븐에 넣어 굽기 전 가위를 사용하여, 윗 엣지 8부분을 살짝 잘라준다

2)구운 후 가위로 자른 부분 양쪽 끝을 맞춘 후 반달칼을 이용하여 잘라준다.

*주의사항

< 참조 >
1. 피자에 치킨 향이 밸 수 있으므로 기름 받이 물을 자주 갈아줄 것 2. 종이 호일을 깔고 조리 시 피자 도우의 조리가 덜 될 수 있고 밑면이 눅눅해질 수 있음 3. 피자를 상단에 조리하여 바람에 치즈가 날려 치킨 위에 떨어질 수 있으므로 주의 필요 (아주 소량이고 미미하지만 한 번 더 확인이 필요) 4. 치즈가 굳을 시 재조리 하여도 녹지 않고 도우가 딱딱해질 수 있으니 재조리 금지 5. 유산지 위에서 피자를 커팅 할 경우 종이가 잘려 피자에 혼입 우려 있으므로 금지 6. 지형이 험한 곳으로 배달 시 피자가 쏠릴 수 있으므로 삼발이 세 개 정도로 고정 추천 7. 피자 해동 시 해동 모드가 아닌 조리 모드 사용 시 과 조리 되어 품질이 저하될 수 있으므로 해동모드 매뉴얼 준수하여 해동할 것
