1. 주재료

1) 절단계육 (냉장보관)
포장단위 : 955g이상/봉
청구단위 : 봉
유통기한 : 제조일 포함 6일 (유통기한 표시일)

묶음배송

2) 굽네 불금치킨 소스 (냉장보관)
포장단위 : 75gx50ea 1봉/ 2봉 / box
청구단위 : 박스
유통기한 : 제조일로부터 8개월

3) 굽네 불맛 풍미유 (실온보관)
포장단위 : 10g x50ea x1봉
청구단위 : 봉
유통기한 : 제조일로부터 12개월

4) 버터갈릭 소스 (냉장보관)
포장단위 : 500g pack
청구단위 : 1pack
유통기한 : 제조일로 부터 냉장 3개월

5) 굽네 오매불맛 소스 (냉장보관)
포장단위 : 30gx100ea 1봉
청구단위 : 봉
유통기한 : 제조일로부터 8개월

2. 기구 및 장비

(1) 면장갑 (2) 집게 (3) 가위 (4) 위생장갑 (5) 믹싱볼 (6) 그리드 (7) 타이머

3. 제조 (굽기)

1) 유통기한 및 선입선출을 확인한 후 가위로 개봉하여 그리드에 붓는다.
* 남아 있는 시즈닝, 이물질 제거 및 확인
* 결수 확인 (다리2개, 윙2개, 봉2개, 목1개)

2) 다리를 집게로 잡고 종아리 쪽을 가위로 뼈가 보이도록 절단해 펴준다.

3) 목에 있는 지방은 완벽히 제거한다.

4) 가슴살과 안심이 붙어 있는 두꺼운 부분은 넓게 펴준다.

5) 그리드 위에 원료육이 서로 붙지 않게하고, 스킨이 위로 향하도록 정리한다.

6) [1차 굽기]
오븐의 온도가 195℃가 되어있는지 확인 후,
그리드를 넣고 12분간 (±1분) 굽는다.
* 오븐 문은 신속하게 열고 닫는다.

7) [1차 소스]
타이머가 울리면 치킨을 꺼낸 후,
굽네 불금치킨 소스 1봉(75g) 을 믹싱볼에 담고 소스가
고르게 묻을 수 있도록 충분히 텀블링 한다.
* 떡볶이 떡, 그릴후랑크 추가 시 굽네 오매불맛 소스30g을
추가하여 조리 한다.

8) [2차 굽기]
그리드에 종이 호일을 깔고(7)의 치킨을 서로 붙지 않게
하고 스킨이 위로 향하도록 정리한 후 4분간 (±1분)
굽는다.
* 다른 치킨의 기름과 이물질이 떨어 지지 않도록 상단에서
조리한다.
* 과 조리로 인해 치킨표면의 소스가 타지 않도록 주의한다.

9) 2차 굽기가 완료 되면 치킨을 꺼낸 후
제품이 손상이 가지 않도록 주의 해서 검게 탄 부분은
가위를 이용하여 제거한다.
* 제거부위 : 소스가 검게 탄 가장자리 부분

10) [2차 소스]
① 굽네 불금치킨 소스 1봉(75g)와 불맛 풍미유 (10g)를
믹싱볼에 담고 고무주걱을 이용하여 가볍게 섞어 놓는다.

② 섞어놓은 소스에 (9)의 치킨을 넣어 소스가 고르게
묻을 수 있도록 충분히 텀블링 한다.

* 불금치킨 소스와 불맛풍미유는 반드시 섞은후 텀블링한다.
섞지 않고 텀블링 시 쓴맛과 기름진 맛이 나며
전체적인 맛의 조화가 떨어지므로 충분히 섞어 텀블링 한다.

11) 굽네 트레이에 가지런히 담는다.
* 믹싱볼에 남은 소스는 실리콘 고무주걱을 이용 하여 치킨
위에 모두 올린다.

12) 버터갈릭 소스40g (20줄이상)을 지그재그로 뿌려준다.
* 소스는 치킨 맨 위에만 도포하기
(치킨 안쪽에 도포하지 않음)
* 소스통에 담아 사용,지름0.5cm
* 소분 사용 시 소스통에 담긴 제품 사용기한 7일
* 소스통에 소분하고 팩에 남겨진 소스는 30일 이내 사용

13) 굽네 박스에 담고 각무와 굽네 오매불맛 디핑소스30g 1봉을 함께 제공한다.

* 홀 플레이팅
