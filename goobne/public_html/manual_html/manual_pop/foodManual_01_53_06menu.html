1. 주재료

1) 북채 (냉장보관)
포장단위 : 905g이상
청구단위 : 봉
유통기한 : 유통기한 표시일
2) 통 날개(냉장보관)
포장단위 : 930g이상/봉
청구단위 : 봉
유통기한 : 유통기한 표시일

묶음배송

2) 굽네 불금치킨 소스 (냉장보관)
포장단위 : 75gx50ea 1봉/ 2봉 / box
청구단위 : 박스
유통기한 : 제조일로부터 8개월

3) 굽네 불맛 풍미유 (실온보관)
포장단위 : 10g x50ea x1봉
청구단위 : 봉
유통기한 : 제조일로부터 12개월

4) 버터갈릭 소스 (냉장보관)
포장단위 : 500g pack
청구단위 : 1pack
유통기한 : 제조일로 부터 냉장 3개월

5) 굽네 오매불맛 소스 (냉장보관)
포장단위 : 30gx100ea 1봉
청구단위 : 봉
유통기한 : 제조일로부터 8개월

2. 기구 및 장비

(1) 면장갑 (2) 집게 (3) 가위 (4) 그리드 (5) 오븐장갑 (6) 믹싱볼 (7) C팬 (8) 위생장갑
(9) 실리콘주걱 (10) 종이호일

3. 제조 (굽기)

1) 유통기한 및 선입선출을 확인 한 후 가위로 개봉하여
북채½, 통 날개 ½ 의 분량을 그리드에 붓는다.
* 북채(905g이상)-10개~12개, 통 날개(930g) : 12개~13개
[기준]: 북채 10개, 통 날개 12개 – 북채5개,통 날개 6개 사용
예시 1) 북채11개 통날개12개
<1> 6개 5.5개
<2> 5개 6.5개
예시 2) 북채11개 통날개13개
<1> 5개 7개
<2> 6개 6개
예시 3) 북채12개 통날개12개
<1> 6개 6개
<2> 6개 6개
예시 4) 북채12개 통날개13개
<1> 6개 6.5개
<2> 6개 6.5개
각 부분 육을 1/2씩 혼합 하였을 때 약920g 이상 중량이
될 수 있도록 분할 하여 사용
※ 북채와 통 날개 ½ 남은 분량은 밀봉 하여 냉장보관
※ 당일소진

2) 통날개
: 가위를 이용하여 날개 끝 뾰족한 부분까지 잘라낸다.
: 통 날개 중간관절(연골) 부위를 자른다.
※ 뼈 부분은 절단하면 조리 후 뼈 부분이 변색됨
※ 윙,봉은 조리 전에 절단하여 연골까지 익힘

3) 북채
: 다리를 집게로 잡고 종아리 쪽을 가위로 뼈가 보이도록
절단해 펴준다.
: 다리의 자른 단면 양쪽을 약2mm 절개 한다.

4) 그리드 위에 집게를 이용하여 원료육이 서로 붙지 않게
하고 껍질이 위로 향하도록 정리한다.
※ 북채의 스킨을 최대한 앞으로 당긴다.
※ 북채를 그리드의 가로 방향으로 다리를 올린다.

5) 오븐의 온도가 195℃가 되어 있는지 확인한 후 오븐 문을
열고 그리드를 넣은 후 문을 닫고 12분간 (±1분)
굽는다.
※ 윙이 그리드 사이로 떨어지지 않도록 주의!

6 ) [1차소스]
타이머가 울리면 치킨을 꺼낸 후,
굽네 불금치킨 소스 1봉(75g)을 믹싱볼에 담고 소스가
고르게 묻을 수 있도록 충분히 텀블링 한다.
떡볶이 떡, 그릴후랑크 추가 시 굽네 오매불맛 소스30g을
추가하여 조리 한다.

7 ) [2차굽기]
그리드에 종이 호일을 깔고(7)의 치킨을 서로 붙지 않게
하고 스킨이 위로 향하도록 정리한 후 4분간 (±1분)
굽는다.
※ 다른 치킨의 기름과 이물질이 떨어 지지 않도록 상단에서
조리한다.
※ 과 조리로 인해 치킨표면의 소스가 타지 않도록 주의한다.

8) 2차 굽기가 완료 되면 치킨을 꺼낸 후
제품이 손상이 가지 않도록 주의 해서 검게 탄 부분은
가위를 이용하여 제거한다.
※ 제거부위 : 소스가 검게 탄 가장자리 부분

9) [2차 소스]
① 굽네 불금치킨 소스 1봉(75g)와 불맛 풍미유 (10g)를
믹싱볼에 담고 고무주걱을 이용하여 가볍게 섞어 놓는다.

② 섞어놓은 소스에 (8)의 치킨을 넣어 소스가 고르게
묻을 수 있도록 충분히 텀블링 한다.

* 불금치킨 소스와 불맛풍미유는 반드시 섞은후 텀블링한다.
섞지 않고 텀블링 시 쓴맛과 기름진 맛이 나며
전체적인 맛의 조화가 떨어지므로 충분히 섞어 텀블링 한다.

10) 굽네 트레이에 가지런히 담는다.
* 믹싱볼에 남은 소스는 실리콘 고무주걱을 이용 하여 치킨
위에 모두 올린다.

11) 버터갈릭 소스40g (20줄이상)을 지그재그로 뿌려준다.
* 소스는 치킨 맨 위에만 도포하기
(치킨 안쪽에 도포하지 않음)
* 소스통에 담아 사용,지름0.5cm
* 소분 사용 시 소스통에 담긴 제품 사용기한 7일
* 소스통에 소분하고 팩에 남겨진 소스는 30일 이내 사용

12) 굽네 박스에 담고 각무와 굽네 오매불맛 디핑소스30g 1봉을 함께 제공한다.
